# 메모 (soksak-plugin-memo)

프로젝트별 메모 패널. 우측 사이드바에 메모장(✎)을 띄우고, 입력하면 자동 저장한다.

- **프로젝트별 분리**: 저장 키를 프로젝트 루트 경로에서 파생한다(비안전 문자 `_` 치환 + 해시 접미). 다른 루트의 프로젝트는 서로 다른 키를 쓰므로 메모가 섞이지 않는다. 루트가 없는 컨텍스트는 `memo.global` 키 하나를 공유한다.
- **재시작 후 유지**: 전용 저장소(`~/.soksak/plugins-data/soksak-plugin-memo/`) 파일 기반이라 앱을 껐다 켜도 남는다.
- **자동 저장**: 입력 후 400ms 디바운스로 저장. 상태줄에 "저장 중…/저장됨"을 표시하고, 실패는 빨간 에러 텍스트로 정직하게 노출한다. 패널 언마운트·플러그인 비활성화 시 대기 중 저장을 플러시한다.

## 권한 근거

| 권한 | 근거 |
| --- | --- |
| `ui` | 사이드바에 메모 뷰(`panel`)를 등록·표시 |
| `storage` | 메모 텍스트를 플러그인 전용 저장소에 읽기/쓰기 |

그 외 권한은 선언하지 않는다 — 파일시스템·명령·네트워크 접근 없음.

## 설치

```bash
# GitHub 단축형
sok plugin.install '{"source":"<user>/soksak-plugin-memo"}'

# 로컬 경로(이 예제 디렉토리)
sok plugin.install '{"source":"/path/to/repo/examples/plugins/soksak-plugin-memo"}'

# 활성화(동의는 앱 UI 에서 사람이 직접)
sok plugin.enable '{"id":"soksak-plugin-memo"}'
```

## 사용법

1. 우측 사이드바 아이콘 레일에서 ✎(메모)를 연다.
2. 텍스트 영역에 자유롭게 입력한다 — 400ms 후 자동 저장된다.
3. 프로젝트를 전환하면 해당 프로젝트의 메모가 따로 로드된다.
